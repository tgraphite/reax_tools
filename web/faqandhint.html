常见问题和错误排查提示

返回工具主页

1. Cannot enlarge memory ...

1.1 轨迹过大（如超过数百MB或1GB的情况）
Web版是基于浏览器创造隔离环境运行程序，内存和其他计算资源受到浏览器限制。关闭其他标签页，或缩短轨迹（如4000帧均匀抽400帧）。

1.2 其他问题间接导致的错误，通常是格式问题、文件命名问题（如没有前缀名/后缀名、后缀名错误等）。

1.3 使用离线版，离线版性能远强于Web版，并且没有资源限制，但离线版不会自动作图。

2. 结果内容为空，输出提示中type为空（type 1: , type 2: ,...）

没有指定数字类型对应的元素符号，使用附加参数如-t C,H,O,N指定。或者转为xyz (extended)格式，详见格式说明。

3. 结果不符合预期，形成过多破碎的分子、或过于稠密的大分子。

调整-r参数，详见参数说明。

4. 反应网络结果过于繁杂或过于少。

4.1 -rr选项会抵消互逆反应的其中一支，详见参数说明，使用-rr选项后视图能够更加清晰。

4.2 当反应网络不能全部显示时，会自动分拆为reactions.dot和reactions_full.dot两个文件，作图区显示的前者是后者的高流量部分。另存为所有文件并离线查看和绘制后者。
