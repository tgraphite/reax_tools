-t [元素符号,元素符号...]

值类型：英文逗号分隔的字符串，默认值：无
当使用lammpstrj文件或无元素符号标记的xyz文件时，定义type 1,2,3...对应的元素符号。
务必注意：1.英文逗号，2.参数内无空格。
正确：-t C,H,O,N
错误：-t C，H，O，N (中文逗号)
错误：-t C H O N（应使用且仅使用英文逗号）
错误：-t C, H, O, N（多余的空格）
错误：-t C,H,O（实际有4个type而缺少）
其他类似格式的参数同样注意。

-r [vdW半径缩放因子]

值类型：浮点数，默认值：1.2
van der Waals半径判据的缩放因子。默认值1.2与OVITO create bonds一致，适合一般有机物和无机物体系。
调小这个值会使得判断更严格，容易判断出更小的分子、分子碎片，反之亦然。具体看体系尝试。

-tr [元素符号:半径] [元素符号:半径]...

值类型：字符串和浮点数，默认值：见defines.cpp
设置特定元素的vdW半径，可以使用多次，例如-tr N:1.5 或-tr H:1 N:1.5。
与-r效果叠加。

-tv [元素符号:化合价] [元素符号:化合价]...

值类型：字符串和整数，默认值：见defines.cpp
设置特定元素的化合价，可以使用多次，例如-tv N:4或-tv N:4 S:6。
同时定义-tr和-tv会创建新的元素类型，例如-tr D:1.05 T:1.05 -tv D:1 T:1相当于创建了氢的同位素氘和氚，配合在输入文件里将一些H标记为D和T，可以实现原子的标记和追踪。
使用自定义元素类型时建议使用较小的轨迹先行尝试。

-me [元素符号]

值类型：字符串，默认值：不使用
对物质含量统计，进行按元素含量的归纳。需要与-mr共同使用。
例如，-me C -mr 1,4,8,16会将所有含C的物质归纳到C1-C3、C4-C7、C8-C15、C16+四个组。

-mr [范围值,范围值,范围值...]

值类型：英文逗号分隔的整数，默认值：不使用
元素含量的归纳的范围，左闭右开区间。需要与-me共同使用。
例如，-me C -mr 1,4,8,16会将所有含C的物质归纳到C1-C3、C4-C7、C8-C15、C16+四个组。

-rc

值类型：无，默认值：不使用
进行元素含量归纳后，对物质组的输出数量进行重算，输出该种元素的原子数而不是分子数。
例如，开启-rc后，C16+平均含量由10.5变为250。

--order [元素符号,元素符号...]

值类型：英文逗号分隔的字符串，默认值：有机物相关元素优先（C,H,O,N,S,F,P），其余按原子量。
重命名化学式，按照给定的顺序。
例如，定义顺序为H,N,O，则HO3N变为HNO3。

-norr

值类型：无，默认值：不使用。
不对互逆的反应进行自动抵消。
例如，H2O ⇌ OH，其中H2O → OH = 50, OH → H2O = 40，不开启时，H2O → OH = 10，开启-norr后，保留原本的一对反应。

--max-reactions [数量]

值类型：整数，默认值：60。
reactions.dot默认输出的反应数，也是网页版默认绘制的反应数。
无论如何，如果有更多反应，reactions_full.dot会保存所有。
