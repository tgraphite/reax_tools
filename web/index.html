ReaxTools Pro

ReaxTools Pro simplifies reactive molecule dynamics analysis, offering seamless compatibility with LAMMPS, CP2K, GPUMD, and any program generating XYZ-style trajectories.

The software provides comprehensive visualization tools including:

  • Species, bond, atom type, and ring statistics
  • Advanced network analysis capabilities
  • Interactive reaction network visualization
  • Atom transfer and flow analysis tools
